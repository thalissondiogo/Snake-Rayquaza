## 📌 Sobre
Este é um repositório organizado e mantido por **thalissonsenai-boop**. 
Jogo da cobrinha temático com o Pokémon lendário Rayquaza.

## 🚀 Funcionalidades
- Mecânica clássica do Snake
- Temática customizada (Pokémon)
- Sistema de pontuação

## 🛠️ Tecnologias
Este projeto utiliza:
- **JavaScript, HTML, CSS**

## 💻 Como Rodar
1. Clone este repositório:
   ```bash
   git clone https://github.com/thalissonsenai-boop/Snake-Rayquaza.git
   ```
2. Abra o diretório do projeto:
   ```bash
   cd Snake-Rayquaza
   ```
3. Abra o `index.html` diretamente no seu navegador.
